# Video Chat Backend (King PHP) - Scaffold

This directory is the new backend track for the King-based video-chat runtime.

## Run locally

```bash
cd demo/video-chat/backend-king-php
./run-dev.sh
```

Default endpoint:

- `GET http://127.0.0.1:18080/`

Environment overrides:

- `VIDEOCHAT_KING_HOST` (default `127.0.0.1`)
- `VIDEOCHAT_KING_PORT` (default `18080`)

This is a bootstrap entrypoint for the new stack; functional API/WS contracts are implemented in subsequent V1 issues.
